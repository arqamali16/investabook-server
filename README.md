# Instabook

This is the Server Code for Self Created Instabook Web App

## Quick Start

Clone the Repo

```bash
npm install
npm start
```

# Stuffs Used for building

Libraries/Technologies used are listed below.

- Node 14.0.13
- Nodemon 2.0.3
- Express 4.17.1
- Typescript 3.9.5
- Loopback 4
- Mongo DB 3.5.6
